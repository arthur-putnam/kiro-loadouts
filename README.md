# kiro-loadouts

This repo contains Kiro loadouts that work in conjunction with the kiro-loadouts-extension. This extension provides a way to quickly add or remove sets of Kiro hooks, steering files, tool settings, MCP settings, etc in the form of a "loadout".

This repo contains predefined loadouts, which contains hooks, steering files, tool settings, MCP settings, etc. that are commonly used in development workflows. It is recommended that you make a copy of this repo and add, edit, or remove loadouts to your liking, then configure the extension to point at your own repo of loadouts. These loadouts can help you jumpstart your own project and loadout repo.

If you find loadouts that you think are useful, please contribute to the project by creating a PR with the new loadout or file changes.
